# Diesis-Fidelity

High-fidelity audio operating system based on Debian Bullseye or Bookworm (v11-v12).

Models: TEVERE, AURA, NEPTUNE, etc.

This project provides custom builds and packages for high-quality audio playback using MPD, myMPD, Upmpdcli, and other optimized Debian components.

## Included Components

This distribution contains precompiled binaries from the following open source projects:

- MPD (Music Player Daemon) – GPLv2
- myMPD – GPLv3
- upmpdcli – GPLv2
- Additional Debian Libraries and Utilities.

## About Diesis Audio Players - Diesis-Fidelity

Diesis audio players are high-fidelity music players based on the Diesis-Fidelity operating system. They leverage the optimized MPD stack and custom packages to deliver audiophile-quality playback, prioritizing natural, analog-like sound reproduction.

These players support a wide range of DACs and are designed for use in high-end audio setups, combining the power and flexibility of open source Linux audio software with Diesis-Fidelity's proprietary system optimizations.
