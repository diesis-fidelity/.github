# Diesis-Fidelity

High-Fidelity Audio Operating System based on Debian Bullseye or Bookworm (v11 - v12).

Models: TEVERE, AURA, NEPTUNE, etc.

This project provides custom builds and packages for high-quality audio playback using MPD, myMPD, Upmpdcli, and other optimized Debian components.

## Included Components

This distribution contains precompiled binaries of the following open-source projects:

* MPD (Music Player Daemon) – GPLv2
* myMPD – GPLv3
* upmpdcli – GPLv2
* Additional libraries and utilities from Debian.

## About Diesis Audio Players

Diesis Audio players are high-fidelity music players built on the Diesis-Fidelity OS. They leverage the optimized MPD stack and custom packages to deliver audiophile-grade playback, focusing on natural, analog-like sound reproduction.

These players support a wide range of DACs and are designed for use in high-end audio setups, combining the power and flexibility of open-source Linux audio software with proprietary system optimizations by Diesis-Fidelity.
